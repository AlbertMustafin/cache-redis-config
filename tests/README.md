## Introduction
Cache Redis Config is a software project that provides a simple and efficient way to manage Redis cache configurations.

## Features
* Supports multiple Redis instances
* Automatic cache expiration
* Customizable cache expiration time
* Support for Redis clusters

## Getting Started
To get started with Cache Redis Config, follow these steps:
1. Install the required dependencies: `npm install`
2. Create a new Redis instance: `redis-cli`
3. Configure Cache Redis Config: `config.js`
4. Start the application: `node app.js`

## Configuration
The configuration file is located at `config.js`. The following options are available:
* `redisHost`: The hostname of the Redis instance
* `redisPort`: The port number of the Redis instance
* `cacheExpirationTime`: The time in seconds after which the cache will expire
